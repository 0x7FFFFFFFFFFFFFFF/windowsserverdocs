# Remote Server Administration Tools for Windows Server 2012
The content in this collection describes the tools that help IT pros manage remote computers that are running [!INCLUDE[win8_server_2](../Token/win8_server_2_md.md)].  
  
-   [Remote Server Administration Tools for Windows 8](http://www.microsoft.com/download/en/details.aspx?id=28972) includes [!INCLUDE[sm](../Token/sm_md.md)], Microsoft Management Console \(MMC\) snap\-ins, consoles, [!INCLUDE[wps_2](../Token/wps_2_md.md)] cmdlets and providers, and command\-line tools for managing roles and features that run on [!INCLUDE[win8_server_2](../Token/win8_server_2_md.md)]. In limited cases, the tools can be used to manage roles and features that are running on [!INCLUDE[nextref_server_7](../Token/nextref_server_7_md.md)] or [!INCLUDE[nextref_longhorn](../Token/nextref_longhorn_md.md)]. Some of the tools work for managing roles and features on Windows Server 2003.

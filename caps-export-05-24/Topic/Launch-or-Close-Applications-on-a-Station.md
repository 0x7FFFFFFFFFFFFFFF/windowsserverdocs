# Launch or Close Applications on a Station
As *MultiPoint Dashboard User*, you can open or close an application on a user’s desktop, selected desktops, or all desktops.  
  
#### To launch an application on a user station  
  
1.  In [!INCLUDE[win8_server_multipoint_dashboard](../Token/win8_server_multipoint_dashboard_md.md)], click the thumbnail image of the user desktop on which you want to launch the application, and then, on the **Desktops** tab, click the drop–down menu for **Launch**.  
  
2.  Click **Launch an Application on Selected Desktops**. The **Launch Application** page opens.  
  
3.  Click the application you want to open, or in **Enter an application, folder, document, or Internet resource to open**, type the name of the resource you want to open, and then click **OK**.  
  
#### To launch an application on all user stations  
  
1.  In [!INCLUDE[win8_server_multipoint_dashboard](../Token/win8_server_multipoint_dashboard_md.md)], on the **Desktops** tab, click the drop–down menu for **Launch**.  
  
2.  Click **Launch an Application on All Desktops**. The **Launch Application** page opens.  
  
3.  Click the application you want to open, or in **Enter an application, folder, document, or Internet resource to open**, type the name of the resource you want to open, and then click **OK**.  
  
#### To close an application on a user station  
  
1.  In [!INCLUDE[win8_server_multipoint_dashboard](../Token/win8_server_multipoint_dashboard_md.md)], click the thumbnail image of the desktop on which you want to close an application.  
  
2.  On the **Desktops** tab, click **Close**. The **Close Application** page opens.  
  
3.  Select the name of the application, folder, document, or Internet resource you want to close, and then click **Close Application**.

# Kernel Mode Remote Direct Memory Access (kRDMA)
  
## <a name="BKMK_KRDMA"></a>Kernel Mode Remote Direct Memory Access \(kRDMA\)  
kRDMA is a new feature for [!INCLUDE[win8_server_2](../Token/win8_server_2_md.md)]. Kernel mode Remote Direct Memory Access \(RDMA\) is an accelerated input\-output \(I\/O\) delivery model that allows application software to bypass most software layers to communicate directly with the computer hardware, which improves application performance and reduces delay.  
  
For more information, see the Windows Dev Center topic [kRDMA Reference](http://msdn.microsoft.com/library/hh463974.aspx).

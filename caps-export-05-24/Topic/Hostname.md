# Hostname
Displays the host name portion of the full computer name of the computer. For examples of how this command can be used, see [Examples](assetId:///c6d43992-8243-4f0a-8605-3152c8a8fe9a#BKMK_Examples).  
  
## Syntax  
  
```  
hostname  
```  
  
### Parameters  
  
|Parameter|Description|  
|-------------|---------------|  
|\/?|Displays Help at the command prompt.|  
  
## Remarks  
This command is available only if the Internet Protocol \(TCP\/IP\) protocol is installed as a component in the properties of a network adapter in Network.  
  
## <a name="BKMK_Examples"></a>Examples  
To display the name of the computer, type:  
  
```  
hostname  
```

# What&#39;s New in Windows Server Essentials
The content in this section describes what's new and changed in [!INCLUDE[wseblue_2](../Token/wseblue_2_md.md)]. This content focuses on changes that will potentially have the greatest impact when you use this release.  
  
Windows Server Essentials is now available in two forms:  
  
-   **As a server role**: [!INCLUDE[wseblue_experience](../Token/wseblue_experience_md.md)] is provided as a server role in the Standard and Datacenter editions of [!INCLUDE[winblue_server_2](../Token/winblue_server_2_md.md)]. Installing Windows Server Essentials Experience in [!INCLUDE[winblue_server_2](../Token/winblue_server_2_md.md)] provides you with all the features and functionalities \(such as the Dashboard, client computer backups, and Remote Web Access\) of [!INCLUDE[wseblue_2](../Token/wseblue_2_md.md)] without the locks and limits that are imposed on it. For more information about this new server role, see [Windows Server Essentials Experience Overview &#91;fwlink\_blue&#93;](../Topic/Windows-Server-Essentials-Experience-Overview.md).  
  
-   **As a core Windows Server edition**: [!INCLUDE[wseblue_2](../Token/wseblue_2_md.md)]. As a part of your installation process, you can opt to install [!INCLUDE[wseblue_2](../Token/wseblue_2_md.md)] as a virtual machine by following instructions in a wizard.

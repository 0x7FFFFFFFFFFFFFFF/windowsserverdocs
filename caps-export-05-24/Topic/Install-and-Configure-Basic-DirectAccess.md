# Install and Configure Basic DirectAccess
This overview provides an introduction to the configuration steps required to deploy a single DirectAccess server running [!INCLUDE[winthreshold_server_1](../Token/winthreshold_server_1_md.md)], [!INCLUDE[winblue_server_1](../Token/winblue_server_1_md.md)], or [!INCLUDE[win8_server_1](../Token/win8_server_1_md.md)]with basic settings.  
  
-   [Step 1: Configure the DirectAccess Infrastructure](../Topic/Step-1--Configure-the-DirectAccess-Infrastructure.md). This step includes configuring network and server settings, DNS settings and Active Directory settings.  
  
-   [Step 2: Configure the DirectAccess Server](../Topic/Step-2--Configure-the-DirectAccess-Server.md). This step includes configuring DirectAccess client computers and server settings.  
  
-   [Step 3: Verify Deployments](../Topic/Step-3--Verify-Deployments.md). This includes steps for verifying your deployment.  
  
Before starting your deployment, verify the planning steps described in [Plan a Basic DirectAccess Deployment](../Topic/Plan-a-Basic-DirectAccess-Deployment.md).

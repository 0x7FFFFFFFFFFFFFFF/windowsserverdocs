# Plan a Basic DirectAccess Deployment_1
This topic describes the planning steps required in order to deploy a single DirectAccess server with basic features:  
  
1.  [Step 1: Plan the DirectAccess Infrastructure_2](../Topic/Step-1--Plan-the-DirectAccess-Infrastructure_2.md)—Plan network and server topology, firewall settings, certificate requirements, DNS and Active Directory.  
  
2.  [Step 2: Plan the DirectAccess Deployment_1](../Topic/Step-2--Plan-the-DirectAccess-Deployment_1.md)—Plan client and server deployment.  
  
## Next steps  
After you have completed these planning steps, you can begin the server deployment. For instructions, see [Install and Configure Basic DirectAccess_1](../Topic/Install-and-Configure-Basic-DirectAccess_1.md).

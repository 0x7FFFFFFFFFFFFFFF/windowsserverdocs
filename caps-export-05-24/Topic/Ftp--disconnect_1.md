# Ftp: disconnect_1
Disconnects from the remote computer and retains the **ftp>** prompt. For examples of how this command can be used, see [Examples](assetId:///c6d43992-8243-4f0a-8605-3152c8a8fe9a#BKMK_Examples).  
  
## Syntax  
  
```  
disconnect  
```  
  
### Parameters  
none  
  
## <a name="BKMK_Examples"></a>Examples  
Disconnect from the remote computer and retain the **ftp>** prompt.  
  
```  
disconnect  
```

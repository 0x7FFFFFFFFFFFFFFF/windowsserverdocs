# Install and Configure Basic DirectAccess_1
This overview provides an introduction to the configuration steps required in order to deploy a single DirectAccess server with basic settings.  
  
-   [Step 1: Configure the DirectAccess Infrastructure_](../Topic/Step-1--Configure-the-DirectAccess-Infrastructure_.md). This step includes configuring network and server settings, DNS settings and Active Directory settings.  
  
-   [Step 2: Configure the DirectAccess Server_2](../Topic/Step-2--Configure-the-DirectAccess-Server_2.md). This step includes configuring DirectAccess client computers and server settings.  
  
-   [Step 3: Verify the Deployment_4](../Topic/Step-3--Verify-the-Deployment_4.md). This includes steps for verifying your deployment.  
  
Before starting your deployment, verify the planning steps described in [Plan a Basic DirectAccess Deployment_1](../Topic/Plan-a-Basic-DirectAccess-Deployment_1.md).

# Microsoft Management Console 3.0_deleted
Information about Microsoft Management Console \(MMC\) 3.0 is available on the web. For help getting your server connected to the Internet, see [Get Connected](../Topic/Get-Connected.md). For more information about MMC 3.0, see [Microsoft Management Console 3.0](http://go.microsoft.com/fwlink/p/?linkid=221040).  
  
Microsoft has moved information about this technology to the [Microsoft TechNet website](http://technet.microsoft.com), so that up\-to\-date information is available to you.

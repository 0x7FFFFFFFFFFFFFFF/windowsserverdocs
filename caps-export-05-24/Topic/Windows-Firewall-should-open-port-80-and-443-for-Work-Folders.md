# Windows Firewall should open port 80 and 443 for Work Folders
[!INCLUDE[win8_bpa_intro](../Token/win8_bpa_intro_md.md)]  
  
|||  
|-|-|  
|Operating System|[!INCLUDE[winblue_server_2](../Token/winblue_server_2_md.md)]|  
|Product\/Feature|File and Storage Services|  
|Severity|Error|  
|Category|Configuration|  
  
## Issue  
*Work Folders is enabled, but both port 80 and 443 are closed in Windows Firewall.*  
  
## Impact  
*Users will not be able to sync with this server using Work Folders.*  
  
## Resolution  
*Use Windows Firewall to open port 443 and port 80.*

<tr><td width='100%' colspan='3'><div>After installing updates released March 9, 2021 or March 15, 2021, you might get unexpected results when printing from some apps. Issues might include:</div><ul><li>Elements of the document might print as solid black/color boxes or might be missing, including barcodes, QR codes, and graphics elements, such as logos.</li><li>Table lines might be missing. It is a cumulative update, so you do not need to apply any previous update before installing it. You will need to <strong>Check for updates </strong>to install <a href='https://support.microsoft.com/help/5001634' target='_blank'> KB5001634</a> and might need to also select <strong>Download and Install</strong>. For instructions on how to <strong>Check for updates</strong>, please see <a href="https://support.microsoft.com/windows/update-windows-10-3c5ae7fc-9fb6-9af1-1984-b5e0412c556a" rel="noopener noreferrer" target="_blank">Update Windows 10</a>. You will now need to start at step 1 again.</li><li>Expand <strong>Print Servers</strong>, expand your computer's name, and select <strong>Printers</strong>.</li><li>You can now see the Driver Type for each of your installed printer drivers.</li></ol><div><br></div><div><strong>Workaround: </strong>To mitigate the issue, follow the instructions in <a href="https://www.microsoft.com/videoplayer/embed/RWzqv8" rel="noopener noreferrer" target="_blank">this video</a>. It is a cumulative update, so you do not need to apply any previous update before installing it. You will need to <strong>Check for updates </strong>to install <a href='https://support.microsoft.com/help/5001565' target='_blank'> KB5001565</a> and might need to also select <strong>Download and Install</strong>.

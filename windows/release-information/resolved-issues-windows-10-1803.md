For instructions on how to <strong>Check for updates</strong>, please see <a href="https://support.microsoft.com/windows/update-windows-10-3c5ae7fc-9fb6-9af1-1984-b5e0412c556a" rel="noopener noreferrer" target="_blank">Update Windows 10</a>. <tr><td width='100%' colspan='3'><div>After installing <a href='https://support.microsoft.com/help/5000809' target='_blank'> KB5000809</a>, you might receive an APC_INDEX_MISMATCH error with a blue screen when attempting to print to certain printers in some apps. This issue affects a subset of Type 3 printer drivers and does not affect printer drivers that are Type 4. If you are unsure which type your printer driver is, use the following steps:</div><ol><li>Press the Windows key+r or select <strong>Start </strong>and type <strong>run </strong>and select it.</li><li>Type <strong>printmanagement.msc </strong>and press enter or select the <strong>OK button</strong>.</li><li>If Print Management opens, continue to step 4. If you receive the error, <strong>Windows can not find 'printmanagement.msc'.</strong> then select <strong>Start </strong>and type <strong>manage optional features</strong> and select it. You will now need to start at step 1 again.</li><li>Expand <strong>Print Servers</strong>, expand your computer's name, and select <strong>Printers</strong>.</li><li>You can now see the Driver Type for each of your installed printer drivers.</li></ol><div><br></div><div><strong>Workaround: </strong>To mitigate the issue, follow the instructions in <a href="https://www.microsoft.com/videoplayer/embed/RWzqv8" rel="noopener noreferrer" target="_blank">this video</a>. It is a cumulative update, so you do not need to apply any previous update before installing it. You will need to <strong>Check for updates </strong>to install <a href='https://support.microsoft.com/help/5001565' target='_blank'> KB5001565</a> and might need to also select <strong>Download and Install</strong>. For instructions on how to <strong>Check for updates</strong>, please see <a href="https://support.microsoft.com/windows/update-windows-10-3c5ae7fc-9fb6-9af1-1984-b5e0412c556a" rel="noopener noreferrer" target="_blank">Update Windows 10</a>. You can import this update&nbsp;into Windows Server Update Services (WSUS) manually. See the&nbsp;<a href="https://docs.microsoft.com/windows-server/administration/windows-server-update-services/manage/wsus-and-the-catalog-site#the-microsoft-update-catalog-site" rel="noopener noreferrer" target="_blank">Microsoft Update Catalog</a>&nbsp;for instructions.</div><div align='right'><a href ='#resolved-issues'>Back to top</a></div></td></tr>

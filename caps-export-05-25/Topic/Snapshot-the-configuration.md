# Snapshot the configuration
This completes the Base Configuration test lab. To save this configuration for additional test labs, do the following:  
  
1.  On all physical computers or virtual machines in the test lab, close all windows and then perform a graceful shutdown.  
  
2.  If your lab is based on virtual machines, save a snapshot of each virtual machine and name the snapshots [!INCLUDE[win8_server_1](../Token/win8_server_1_md.md)] VPN Cloud Configuration. If your lab uses physical computers, create disk images to save the Base Configuration.

# DHCP Diagnostics
The following diagnostic tools and procedures are available to assist in monitoring and troubleshooting the DHCP Server and DHCP Client services in Windows Server. This list will be updated periodically when new information is available.  
  
-   DHCP Server Event Logging  
  
    -   [DHCP Server Events](http://technet.microsoft.com/library/cc726909.aspx): DHCP Server events and errors documentation for [!INCLUDE[nextref_longhorn](../Token/nextref_longhorn_md.md)].  
  
    -   [DHCP Server Operational Events](../Topic/DHCP-Server-Operational-Events.md): Information about new operational events added with [!INCLUDE[nextref_server_7](../Token/nextref_server_7_md.md)].  
  
-   DHCP Client Event Logging  
  
    -   [DHCP Client Events](http://technet.microsoft.com/library/cc726865.aspx): DHCP Client events and errors documentation for Windows Vista.  
  
-   DHCP Console Icons  
  
    -   [DHCP Console Icons](http://technet.microsoft.com/library/cc784812(v=ws.10).aspx): Reference for Windows Server 2003 and [!INCLUDE[nextref_longhorn](../Token/nextref_longhorn_md.md)].  
  
    -   [DHCP Console Icons](http://technet.microsoft.com/library/gg722802(v=ws.10).aspx): Reference for [!INCLUDE[nextref_server_7](../Token/nextref_server_7_md.md)].  
  
    -   [DHCP Console Icons Reference](../Topic/DHCP-Console-Icons-Reference.md): Reference for [!INCLUDE[win8_server_2](../Token/win8_server_2_md.md)] and [!INCLUDE[winblue_server_2](../Token/winblue_server_2_md.md)].

# Manage MultiPoint Systems Using MultiPoint Dashboard
In [!INCLUDE[win8_server_multipoint_dashboard](../Token/win8_server_multipoint_dashboard_md.md)], on the **Systems** tab, you can do the following:  
  
### Restart or shut down selected systems  
You can restart or shut down selected MultiPoint Services systems. For information about restarting or shutting down selected systems, see the [Restart or Shut Down MultiPoint Systems](../Topic/Restart-or-Shut-Down-MultiPoint-Systems.md) topic.  
  
### Remap selected systems  
You can remap selected MultiPoint Services systems. For more information about remapping selected systems, see the [Remap Selected MultiPoint Systems](../Topic/Remap-Selected-MultiPoint-Systems.md) topic.

# Bitsadmin cache
  
## Syntax  
  
```  
Bitsadmin /Cache /Help  
Bitsadmin /Cache /Clear  
Bitsadmin /Cache /Delete  
Bitsadmin /Cache /DeleteURL  
Bitsadmin /Cache /List  
Bitsadmin /Cache /Info  
Bitsadmin /Cache /GetLimit  
Bitsadmin /Cache /SetLimit  
Bitsadmin /Cache /GetExpirationTime  
Bitsadmin /Cache /SetExpirationTime  
```  
  
## Parameters  
  
|Parameter|Description|  
|-------------|---------------|  
|[Bitsadmin cache and help](../Topic/Bitsadmin-cache-and-help.md)|Displays the command\-line usage for the \/Cache commands.|  
|[Bitsadmin cache clear](../Topic/Bitsadmin-cache-clear.md)|Purges the local cache.|  
|[Bitsadmin cache and delete](../Topic/Bitsadmin-cache-and-delete.md)|Deletes a cache entry.|  
|[Bitsadmin cache and deleteurl](../Topic/Bitsadmin-cache-and-deleteurl.md)|Deletes all cache entries for the given URL.|  
|[Bitsadmin cache and list](../Topic/Bitsadmin-cache-and-list.md)|Lists all cache entries.|  
|[Bitsadmin cache and info](../Topic/Bitsadmin-cache-and-info.md)|Dumps a specific cache entry.|  
|[Bitsadmin cache and getlimit](../Topic/Bitsadmin-cache-and-getlimit.md)|Retrieves the cache limit.|  
|[Bitsadmin cache and setlimit](../Topic/Bitsadmin-cache-and-setlimit.md)|Sets the cache limit.|  
|[Bitsadmin cache and getexpirationtime](../Topic/Bitsadmin-cache-and-getexpirationtime.md)|Retrieves the cache expiration time.|  
|[Bitsadmin cache and setexpirationtime](../Topic/Bitsadmin-cache-and-setexpirationtime.md)|Sets the cache expiration time.|

# Windows PowerShell Support for Windows Server
Most of the features in Windows Server provide support for [!INCLUDE[wps_1](../Token/wps_1_md.md)] by including modules that enable you to install,  configure, and use the feature at a [!INCLUDE[wps_2](../Token/wps_2_md.md)] prompt in addition to the traditional graphical user interface.  
  
For more information, see the following:  
  
1.  [Windows PowerShell](http://go.microsoft.com/fwlink/p/?linkid=137640)  
  
2.  [Windows PowerShell Core Module Reference](http://go.microsoft.com/fwlink/p/?linkid=308904)  
  
3.  [Windows Server and Client Module Reference](http://go.microsoft.com/fwlink/p/?linkid=308906)

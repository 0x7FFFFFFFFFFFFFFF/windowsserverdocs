# Fveupdate
FveUpdate is an internally used tool that is run by setup when a computer is upgraded from [!INCLUDE[nextref_vista](../Token/nextref_vista_md.md)] or [!INCLUDE[nextref_longhorn](../Token/nextref_longhorn_md.md)] to [!INCLUDE[nextref_client_7](../Token/nextref_client_7_md.md)] or [!INCLUDE[nextref_server_7](../Token/nextref_server_7_md.md)]. It updates the metadata associated with BitLocker to the latest version. This tool cannot be run independently.

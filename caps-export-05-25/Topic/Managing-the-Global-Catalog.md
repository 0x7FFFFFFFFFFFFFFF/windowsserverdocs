# Managing the Global Catalog
  Designate global catalog servers to accommodate users in sites where a global catalog server is required, for example, to accommodate forest\-wide directory searching and to facilitate domain client logons when universal groups are available. For information about global catalog servers, see How the Global Catalog Works \([http:\/\/go.microsoft.com\/fwlink\/?LinkId\=107063](http://go.microsoft.com/fwlink/?LinkId=107063)\).

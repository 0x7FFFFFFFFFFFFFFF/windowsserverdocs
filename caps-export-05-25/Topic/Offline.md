# Offline
Takes an online disk or volume to the offline state.  
  
## Syntax  
  
```  
Offline disk  
Offline volume  
```  
  
## Parameters  
  
|Parameter|Description|  
|-------------|---------------|  
|[Offline disk](../Topic/Offline-disk.md)|Takes the online disk with focus to the offline state.|  
|[Offline volume](../Topic/Offline-volume.md)|Takes the online volume with focus to the offline state.|

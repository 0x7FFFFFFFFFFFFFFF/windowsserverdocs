# Bitsadmin cache and list
Lists all cache entries.  
  
## Syntax  
  
```  
bitsadmin /Cache /List [/Verbose]   
```  
  
## <a name="BKMK_examples"></a>Examples  
The following example lists all cache entries in verbose format.  
  
```  
C:\> bitsadmin /Cache /List /Verbose  
```

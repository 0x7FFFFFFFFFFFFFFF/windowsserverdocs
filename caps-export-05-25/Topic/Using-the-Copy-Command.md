# Using the Copy Command
Copies an image or a driver group.  
  
## Subcommands  
  
|Subcommand|Description|  
|--------------|---------------|  
|[Using the copy-Image Command](../Topic/Using-the-copy-Image-Command.md)|Copies images that are within the same image group.|  
|[Using the copy-DriverGroup Command](../Topic/Using-the-copy-DriverGroup-Command.md)|Copies an existing driver group on the server.|

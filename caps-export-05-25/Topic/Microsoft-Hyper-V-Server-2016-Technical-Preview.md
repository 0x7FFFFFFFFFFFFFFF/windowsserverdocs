# Microsoft Hyper-V Server 2016 Technical Preview
**This is preliminary content and subject to change.**  
  
[!INCLUDE[winthreshold_mshvs_2](../Token/winthreshold_mshvs_2_md.md)] is a stand\-alone product that contains only the Windows hypervisor, a Windows Server driver model, and virtualization components. It provides a simple and reliable virtualization solution to help you improve your server utilization and reduce costs.  
  
The Windows hypervisor technology in [!INCLUDE[winthreshold_mshvs_2](../Token/winthreshold_mshvs_2_md.md)] is the same as what's in the Hyper\-V role on [!INCLUDE[winthreshold_server_2](../Token/winthreshold_server_2_md.md)].   So much of the content available for the Hyper\-V role on [!INCLUDE[winthreshold_server_2](../Token/winthreshold_server_2_md.md)] also applies to [!INCLUDE[winthreshold_mshvs_2](../Token/winthreshold_mshvs_2_md.md)].

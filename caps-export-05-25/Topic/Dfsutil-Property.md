# Dfsutil Property
The **dfsutil property** command displays or modifies the properties of a folder target \(link target\) or namespace server \(root target\).  
  
For examples of how this command can be used, see [Examples](assetId:///c6d43992-8243-4f0a-8605-3152c8a8fe9a#BKMK_Examples).  
  
## Syntax  
  
```  
dfsutil property Sitecosting | RootScalability | ABE | Insite | TargetfailBack | SD | State | TTL | PriorityRank | PriorityClass | Comment | /?  
```  
  
### Parameters  
  
|Parameter|Description|  
|-------------|---------------|  
|[Dfsutil Property Sitecosting](assetId:///236d03dd-3ae4-4794-bbb1-db59a1b6f041)|Displays or modifies site costing for a namespace.|  
|[Dfsutil Property RootScalability](assetId:///c190d613-a571-4c57-90d6-56623eec5e8f)|Displays or modifies the namsespace polling mode.|  
|[Dfsutil Property ABE](assetId:///4218cd88-d61e-4f17-8818-cfbed6448d0c)|Enable\/Disable\/View ABE property of a namespace.|  
|[Dfsutil Property Insite](assetId:///3fb8406c-f239-4194-b4a5-1b39226afc2e)|Displays or modifies the in\-site property.|  
|[DFSutil Property TargetfailBack](assetId:///27c5b08d-6147-4b36-9f98-2ef217ffa891)|Displays or modifies client fail back.|  
|[Dfsutil Property SD](assetId:///79c69323-c43d-4eaf-9b27-552523220b68)|Set\/Get Security Information on the folder.|  
|[Dfsutil Property State](assetId:///97c46e4a-284a-44d0-99a2-82f8fd58351b)|Displays or modifies a folder target or namespace server.|  
|[Dfsutil Property TTL](assetId:///017b3e86-173f-4c16-8fcc-55c20aad7c72)|Displays or changes client referral caching.|  
|[Dfsutil Property PriorityRank](assetId:///1444d050-642d-40fe-8a2b-20dbdf955bb3)|Displays or changes the ordering method \(priority rank\).|  
|[Dfsutil Property PriorityClass](assetId:///0784dea3-1214-4ab5-aea9-61c722a8d191)|Displays or changes the target priority.|  
|[Dfsutil Property Comment](assetId:///6f79ebf7-0b7e-4c3d-8261-03f7b9ee186a)|Sets or displays the comment for a namespace or link.|  
|\/?|Displays help at the command prompt.|

# Using the New Command
Creates capture and discover images, multicast transmissions, and namespaces.  
  
## Subcommands  
  
|Subcommand|Description|  
|--------------|---------------|  
|[Using the new-CaptureImage Command](../Topic/Using-the-new-CaptureImage-Command.md)|Creates a new capture image from an existing boot image.|  
|[Using the new-DiscoverImage Command](../Topic/Using-the-new-DiscoverImage-Command.md)|Creates a new discover image from an existing boot image.|  
|[Using the new-MulticastTransmission Command](../Topic/Using-the-new-MulticastTransmission-Command.md)|Creates a new multicast transmission.|  
|[Using the new-Namespace Command](../Topic/Using-the-new-Namespace-Command.md)|Creates a new namespace.|

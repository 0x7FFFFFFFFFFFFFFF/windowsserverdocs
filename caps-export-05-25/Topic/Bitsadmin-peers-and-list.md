# Bitsadmin peers and list
Lists all peers.  
  
## Syntax  
  
```  
bitsadmin /Peers /List   
```  
  
## <a name="BKMK_examples"></a>Examples  
The following example lists all peers.  
  
```  
C:\>bitsadmin /Peers /List  
```

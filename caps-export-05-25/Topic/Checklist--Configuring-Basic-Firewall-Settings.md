# Checklist: Configuring Basic Firewall Settings
This checklist includes tasks for configuring a GPO with firewall defaults and settings that are separate from the rules.  
  
## <a name="bkmk_section1"></a>![](../Image/2b05dce3-938f-4168-9b8f-1f4398cbdb9b.gif)**Checklist: Configuring firewall defaults and settings**  
  
||Task|Reference|  
|-|--------|-------------|  
|![](../Image/WFAS_icon_checkbox.gif)|Turn the firewall on and set the default inbound and outbound behavior.|![](../Image/15dd35b6-6cc6-421f-93f8-7109920e7144.gif)[Turn on Windows Firewall and Configure Default Behavior](../Topic/Turn-on-Windows-Firewall-and-Configure-Default-Behavior.md)|  
|![](../Image/WFAS_icon_checkbox.gif)|Configure the firewall to not display notifications to the user when a program is blocked, and to ignore locally defined firewall and connection security rules.|![](../Image/15dd35b6-6cc6-421f-93f8-7109920e7144.gif)[Configure Windows Firewall to Suppress Notifications When a Program Is Blocked](../Topic/Configure-Windows-Firewall-to-Suppress-Notifications-When-a-Program-Is-Blocked.md)|  
|![](../Image/WFAS_icon_checkbox.gif)|Configure the firewall to record a log file.|![](../Image/15dd35b6-6cc6-421f-93f8-7109920e7144.gif)[Configure the Windows Firewall Log](../Topic/Configure-the-Windows-Firewall-Log.md)|

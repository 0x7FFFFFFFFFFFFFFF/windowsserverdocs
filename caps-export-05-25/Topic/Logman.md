# Logman
**Logman** creates and manages Event Trace Session and Performance logs and supports many functions of Performance Monitor from the command line.  
  
## Syntax  
  
```  
logman [create | query | start | stop | delete| update | import | export | /?] [options]  
```  
  
## Actions  
  
|Action|Description|  
|----------|---------------|  
|[logman create](../Topic/logman-create.md)|Create a counter, trace, configuration data collector, or API.|  
|[logman query](../Topic/logman-query.md)|Query data collector properties.|  
|[logman start &#124; stop](../Topic/logman-start---stop.md)|Start or stop data collection.|  
|[logman delete](../Topic/logman-delete.md)|Delete an existing data collector.|  
|[logman update](../Topic/logman-update.md)|Update the properties of an existing data collector.|  
|[logman import &#124; export](../Topic/logman-import---export.md)|Import a data collector set from an XML file or export a data collector set to an XML file.|

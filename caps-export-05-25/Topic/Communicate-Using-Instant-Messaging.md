# Communicate Using Instant Messaging
As a MultiPoint Dashboard User, you can communicate in real\-time with other users by using the instant messaging \(IM\) feature.  
  
#### To chat with a user or multiple users using instant messaging  
  
1.  In [!INCLUDE[win8_server_multipoint_dashboard](../Token/win8_server_multipoint_dashboard_md.md)], on the **Home** tab, click the thumbnail image or images of the user or users with whom you want to chat.  
  
2.  On the **IM** tab, click **Send**. An IM chat window opens.  
  
3.  Type your message, and then click **Send**.

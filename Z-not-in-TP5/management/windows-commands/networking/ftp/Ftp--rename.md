# Ftp: rename
Renames remote files. For examples of how this command can be used, see [Examples](assetId:///c6d43992-8243-4f0a-8605-3152c8a8fe9a#BKMK_Examples).  
  
## Syntax  
  
```  
rename <FileName> <NewFileName>  
```  
  
### Parameters  
  
|Parameter|Description|  
|-------------|---------------|  
|<FileName>|Specifies the file that you want to rename.|  
|<NewFileName>|Specifies the new file name.|  
  
## <a name="BKMK_Examples"></a>Examples  
Rename the remote file **example.txt** to **example1.txt**  
  
```  
Rename example.txt example1.txt  
```

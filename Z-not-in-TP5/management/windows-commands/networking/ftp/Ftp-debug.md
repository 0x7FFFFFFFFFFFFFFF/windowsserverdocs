# Ftp:debug
Toggles Debugging Mode. For examples of how this command can be used, see [Examples](assetId:///c6d43992-8243-4f0a-8605-3152c8a8fe9a#BKMK_Examples).  
  
## Syntax  
  
```  
debug  
```  
  
### Parameters  
none  
  
## Remarks  
  
-   By default, debugging is off.  
  
-   When debugging is on, each command that is sent to the remote computer is displayed, preceded by the greater than (>) character.  
  
## <a name="BKMK_Examples"></a>Examples  
Toggle debug mode on and off.  
  
```  
debug  
```

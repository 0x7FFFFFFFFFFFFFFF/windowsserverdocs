# WSE_Migrate2008.cover
## Migrating Windows Small Business Server 2008 to Windows Server Essentials  

  
### Abstract  
 This guide describes how to migrate an existing  Windows Server Essentials domain to new or significantly modified computer hardware, and then to migrate the settings and data. This guide also describes how to remove your existing server from the  Windows Server Essentials network after you finish the migration.

# WSE_LogCollector.cover
## Using the Windows Server Essentials Log Collector  
 Microsoft Corporation    
  
### Abstract  
 This document describes how to install and run the  Windows Server Essentials Log Collector. The  Windows Server Essentials Log Collector obtains data from multiple sources that is used by Microsoft Customer Service and Support to assist you in solving problems that occur with your server and computers that are connected to the network.  
  
 To view the most recent version of this guide, see [Using the Windows Server Essentials Log Collector](http://go.microsoft.com/fwlink/?LinkId=255211) at the Microsoft TechNet website.

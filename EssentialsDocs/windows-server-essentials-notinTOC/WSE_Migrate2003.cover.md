# WSE_Migrate2003.cover
## Migrating Windows Small Business Server 2003 to Windows Small Business Server 8 Beta  
 Microsoft Corporation  
  
### Abstract  
 This guide explains how to install  Windows Server Essentials in migration mode on a new server, and then migrate the settings and data from the old server that is running Windows Small Business Server 2003 to the new server that is running  Windows Server Essentials. This guide also helps you demote and remove your old server from the network after you finish the migration process.

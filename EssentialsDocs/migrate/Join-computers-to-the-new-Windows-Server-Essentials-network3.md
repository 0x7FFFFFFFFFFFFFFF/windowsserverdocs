# Join computers to the new Windows Server Essentials network3
The next step in the migration process is to join client computers to the new  Windows Server Essentials network.  
  
### To join a computer to the  Windows Server Essentials network  
  
-   From a client computer, browse to **http://***destination-servername***/connect** and follow the onscreen instructions.  
  
    > [!NOTE]
    >  The installation process is the same for domain-joined or non-domain-joined client computers.

# Ftp: delete
Deletes files on remote computers. For examples of how this command can be used, see [Examples](assetId:///c6d43992-8243-4f0a-8605-3152c8a8fe9a#BKMK_Examples).  
  
## Syntax  
  
```  
delete <RemoteFile>  
```  
  
### Parameters  
  
|Parameter|Description|  
|-------------|---------------|  
|<RemoteFile>|Specifies the file to delete.|  
  
## <a name="BKMK_Examples"></a>Examples  
Delete the file test.txt on the remote computer.  
  
```  
delete test.txt  
```

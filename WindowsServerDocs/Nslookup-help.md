# Nslookup help
Displays a short summary of **nslookup** subcommands.  
  
## Syntax  
  
```  
{help | ?}  
```  
  
## Parameters  
  
|Parameter|Description|  
|-------------|---------------|  
|{help &#124; ?}|Displays a short summary of **nslookup** subcommands.|

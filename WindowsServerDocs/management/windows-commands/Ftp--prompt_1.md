# Ftp: prompt_1

>Applies To: Windows Server&reg; 2016, Windows Server&reg; 2012 R2, Windows Server&reg; 2012

Toggles between **prompt** mode on and off. For examples of how this command can be used, see [Examples](assetId:///c6d43992-8243-4f0a-8605-3152c8a8fe9a#BKMK_Examples).  
## Syntax  
```  
prompt  
```  
### Parameters  
none  
## Remarks  
-   By default, **prompt** is on.  
-   **FTP** prompts during multiple file transfers to allow you to selectively retrieve or store files.  **Mget** and **mput** transfer all files if **prompt** is off.  
## <a name="BKMK_Examples"></a>Examples  
Toggle prompt mode on and off.  
```  
prompt  
```

# Ftp: bell_1
Toggles an audible sound to occur after each file transfer command is completed. For examples of how this command can be used, see [Examples](assetId:///c6d43992-8243-4f0a-8605-3152c8a8fe9a#BKMK_Examples).  
## Syntax  
```  
bell  
```  
### Parameters  
none  
## Remarks  
By default, bell is off.  
## <a name="BKMK_Examples"></a>Examples  
Toggle an audible sound to occur after each file transfer command is completed.  
```  
bell  
```

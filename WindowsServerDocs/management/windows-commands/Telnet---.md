# Telnet: ?
Displays Telnet Help. For examples of how this command can be used, see [Examples](assetId:///c6d43992-8243-4f0a-8605-3152c8a8fe9a#BKMK_Examples).  
## Syntax  
```  
?  
```  
### Parameters  
none  
## Remarks  
You can also type **h** for Help.  
## <a name="BKMK_Examples"></a>Examples  
Display Telnet Help.  
```  
?  
```

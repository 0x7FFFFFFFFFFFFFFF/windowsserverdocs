# Ftp: bye
Ends the FTP session with the remote computer and exits. For examples of how this command can be used, see [Examples](assetId:///c6d43992-8243-4f0a-8605-3152c8a8fe9a#BKMK_Examples).  
## Syntax  
```  
bye  
```  
### Parameters  
none  
## Remarks  
**Bye** and **quit** both end the FTP session with the remote computer and exits.  
## <a name="BKMK_Examples"></a>Examples  
End the FTP session with the remote computer and exit.  
```  
bye  
```

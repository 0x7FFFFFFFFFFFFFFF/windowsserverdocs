# Bitsadmin cache and getlimit
Retrieves the cache limit.

## Syntax

```
bitsadmin /Cache /GetLimit 
```

## <a name="BKMK_examples"></a>Examples
The following example retrieves the cache limit..

```
C:\>bitsadmin /Cache /GetLimit 
```

# bitsadmin cache clear
Purges the local cache.

## Syntax

```
bitsadmin /Cache /Clear 
```

## <a name="BKMK_examples"></a>Examples
The following example purges the local cache.

```
C:\>bitsadmin /Cache /Clear 
```

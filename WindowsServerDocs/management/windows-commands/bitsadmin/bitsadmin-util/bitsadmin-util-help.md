# bitsadmin util and help
Displays the command\-line usage for the **Util** commands.

## Syntax

```
bitsadmin /Util /help 
```

## <a name="BKMK_examples"></a>Examples
The following command displays the command\-line help for the **Util** commands.

```
bitsadmin /Util /help
```

# bitsadmin peers

```
bitsadmin /Peers /help
bitsadmin /Peers /Discover
bitsadmin /Peers /Clear
bitsadmin /Peers /list
```

## Parameters

|Parameter|Description|
|-------------|---------------|
|[bitsadmin peers and help](bitsadmin-peers/bitsadmin-peers-help.md)|Displays the command\-line usage for the \/Peers commands.|
|[bitsadmin peers and discover](bitsadmin-peers/bitsadmin-peers-discover.md)|Discovers peers again.|
|[bitsadmin peers and clear](bitsadmin-peers/bitsadmin-peers-clear.md)|Clears the peers list.|
|[bitsadmin peers and list](bitsadmin-peers/bitsadmin-peers-list.md)|list all peers.|

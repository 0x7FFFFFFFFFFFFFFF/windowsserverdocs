# Ftp: pwd_1
Displays the current remote computer directory. For examples of how this command can be used, see [Examples](assetId:///c6d43992-8243-4f0a-8605-3152c8a8fe9a#BKMK_Examples).  
## Syntax  
```  
pwd  
```  
### Parameters  
none  
## <a name="BKMK_Examples"></a>Examples  
Display the current remote computer directory.  
```  
pwd  
```

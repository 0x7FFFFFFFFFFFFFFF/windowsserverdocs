# Using the copy Command
Copies an image or a driver group.

## Subcommands

|Subcommand|Description|
|--------------|---------------|
|[Using the copy-Image Command](using-copy-command/using-copyimage-command.md)|Copies images that are within the same image group.|
|[Using the copy-DriverGroup Command](using-copy-command/using-copydrivergroup-command.md)|Copies an existing driver group on the server.|

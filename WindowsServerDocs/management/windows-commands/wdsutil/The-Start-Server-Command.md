# The Start Server Command
Starts the multicast transmission, namespace, Transport Server or all services on the Windows Deployment Services server.

## Subcommands

|Subcommand|Description|
|--------------|---------------|
|[Subcommand: start-MulticastTransmission](the-start-server-command/Subcommand--start-MulticastTransmission.md)|Starts a transmission.|
|[Subcommand: start-Namespace](the-start-server-command/Subcommand--start-Namespace.md)|Starts a namespace.|
|[Subcommand: start-Server](the-start-server-command/Subcommand--start-Server.md)|Starts all services on a Windows Deployment Services server.|
|[Subcommand: start-TransportServer](the-start-server-command/Subcommand--start-TransportServer.md)|Starts all services on a Transport Server.|

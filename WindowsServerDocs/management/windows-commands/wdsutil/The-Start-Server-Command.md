# The start Server Command
starts the multicast transmission, namespace, Transport Server or all services on the Windows deployment Services server.

## Subcommands

|Subcommand|Description|
|--------------|---------------|
|[Subcommand: start-MulticastTransmission](the-start-server-command/subcommand-startmulticasttransmission.md)|starts a transmission.|
|[Subcommand: start-Namespace](the-start-server-command/subcommand-startnamespace.md)|starts a namespace.|
|[Subcommand: start-Server](the-start-server-command/subcommand-startserver.md)|starts all services on a Windows deployment Services server.|
|[Subcommand: start-TransportServer](the-start-server-command/subcommand-starttransportserver.md)|starts all services on a Transport Server.|

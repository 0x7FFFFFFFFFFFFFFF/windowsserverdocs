# Using the New Command
creates capture and discover images, multicast transmissions, and namespaces.

## Subcommands

|Subcommand|Description|
|--------------|---------------|
|[Using the new-CaptureImage Command](using-new-command/using-newcaptureimage-command.md)|creates a new capture image from an existing boot image.|
|[Using the new-DiscoverImage Command](using-new-command/using-newdiscoverimage-command.md)|creates a new discover image from an existing boot image.|
|[Using the new-MulticastTransmission Command](using-new-command/using-newmulticasttransmission-command.md)|creates a new multicast transmission.|
|[Using the new-Namespace Command](using-new-command/using-newnamespace-command.md)|creates a new namespace.|

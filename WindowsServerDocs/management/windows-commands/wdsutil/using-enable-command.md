# Using the Enable Command
Enables all services for Windows deployment Services.

## Subcommands

|Subommand|Description|
|-------------|---------------|
|[Using the enable-Server Command](using-enable-command/using-enableserver-command.md)|Enables all services on a specified Windows deployment Services server \(deployment Server\).|
|[Using the enable-TransportServer Command](using-enable-command/using-enabletransportserver-command.md)|Enables all services on a specified Transport Server.|

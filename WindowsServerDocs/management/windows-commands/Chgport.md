# Chgport
Lists or changes the COM port mappings to be compatible with MS\-DOS applications.

> [!NOTE]
> [!INCLUDE[rd_note-cmd-line-ref](includes/rd_note-cmd-line-ref_md.md)]

## Remarks
This command is the same as the **change port** command.

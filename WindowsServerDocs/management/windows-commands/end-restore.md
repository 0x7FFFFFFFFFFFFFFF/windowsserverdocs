# End restore

>Applies To: Windows Server&reg; 2016, Windows Server&reg; 2012 R2, Windows Server&reg; 2012

Ends a restore session and issues a **PostRestore** event to involved writers.  
  
## Syntax  
  
```  
end restore  
```

# Ftp: mkdir
Creates a directory on the remote computer. For examples of how this command can be used, see [Examples](assetId:///c6d43992-8243-4f0a-8605-3152c8a8fe9a#BKMK_Examples).  
## Syntax  
```  
mkdir <Directory>  
```  
### Parameters  
|Parameter|Description|  
|-------------|---------------|  
|<Directory>|Specifies the name of the new remote directory.|  
## <a name="BKMK_Examples"></a>Examples  
Create a directory called **Dir1** on the remote computer.  
```  
mkdir Dir1  
```

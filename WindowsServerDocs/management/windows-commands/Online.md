# online
Takes an offline disk or volume to the online state.

## Syntax

```
offline disk
offline volume
```

## Parameters

|Parameter|Description|
|-------------|---------------|
|[offline disk](offline/offline-disk.md)|Takes the offline disk with focus to the online state.|
|[offline volume](offline/offline-volume.md)|Takes the offline volume with focus to the online state.|

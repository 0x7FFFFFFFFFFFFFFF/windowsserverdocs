# Ftp: hash_1
Toggles number sign (#) printing for each data block that is transferred. For examples of how this command can be used, see [Examples](assetId:///c6d43992-8243-4f0a-8605-3152c8a8fe9a#BKMK_Examples).  
## Syntax  
```  
hash  
```  
### Parameters  
none  
## Remarks  
-   The size of a data block is 2048 bytes.  
-   By default, **hash** is off.  
## <a name="BKMK_Examples"></a>Examples  
Toggle number sign (#) printing for each data block that is transferred.  
```  
hash  
```

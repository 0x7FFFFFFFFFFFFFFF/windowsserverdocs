# Begin backup

>Applies To: Windows Server&reg; 2016, Windows Server&reg; 2012 R2, Windows Server&reg; 2012

Starts a full backup session.  
  
## Syntax  
  
```  
begin backup  
```  
  
## Remarks  
  
-   This command overrides the default copy backup setting.

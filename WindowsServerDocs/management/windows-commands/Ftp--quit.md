# Ftp: quit
Ends the FTP session with the remote computer and exits **ftp** to the operating system command prompt. For examples of how this command can be used, see [Examples](assetId:///c6d43992-8243-4f0a-8605-3152c8a8fe9a#BKMK_Examples).  
## Syntax  
```  
quit  
```  
### Parameters  
none  
## <a name="BKMK_Examples"></a>Examples  
End the FTP session with the remote computer and exit **ftp** to the operating system command prompt.  
```  
quit  
```

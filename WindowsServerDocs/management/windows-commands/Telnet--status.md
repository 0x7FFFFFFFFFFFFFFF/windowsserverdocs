# Telnet: status

>Applies To: Windows Server&reg; 2016, Windows Server&reg; 2012 R2, Windows Server&reg; 2012

Displays status information.   
## Syntax  
```  
st[atus]  
```  
### Parameters  
none  
## <a name="BKMK_Examples"></a>Examples  
Display status information.  
```  
st  
```

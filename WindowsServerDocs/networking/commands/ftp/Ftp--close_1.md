# Ftp: close_1
Ends the FTP session with the remote server and remains at the **ftp>** prompt. For examples of how this command can be used, see [Examples](assetId:///c6d43992-8243-4f0a-8605-3152c8a8fe9a#BKMK_Examples).  
  
## Syntax  
  
```  
close  
```  
  
### Parameters  
none  
  
## <a name="BKMK_Examples"></a>Examples  
End the FTP session with the remote server and remain at the **ftp>** prompt.  
  
```  
close  
```

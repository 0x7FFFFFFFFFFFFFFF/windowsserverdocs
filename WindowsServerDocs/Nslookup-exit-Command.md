# Nslookup exit Command
Exits **nslookup**.  
  
## Syntax  
  
```  
Nslookup /exit  
```  
  
## Parameters  
  
|Parameter|Description|  
|-------------|---------------|  
|{help &#124; ?}|Displays a short summary of **nslookup** subcommands.|

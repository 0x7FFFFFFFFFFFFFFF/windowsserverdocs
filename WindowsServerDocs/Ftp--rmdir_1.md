# Ftp: rmdir_1
Deletes a remote directory. For examples of how this command can be used, see [Examples](assetId:///c6d43992-8243-4f0a-8605-3152c8a8fe9a#BKMK_Examples).  
  
## Syntax  
  
```  
rmdir <Directory>  
```  
  
### Parameters  
  
|Parameter|Description|  
|-------------|---------------|  
|<Directory>|Specifies the name of the remote directory to delete.|  
  
## <a name="BKMK_Examples"></a>Examples  
Delete the **Pictures** remote directory.  
  
```  
rmdir Pictures  
```

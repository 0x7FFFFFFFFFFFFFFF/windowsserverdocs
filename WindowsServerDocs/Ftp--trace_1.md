# Ftp: trace_1
Toggles packet tracing.

## Syntax

```
trace
```

### Parameters
none

## Remarks
**Trace** displays the series of internal FTP function calls when running an **ftp** command.

## <a name="BKMK_Examples"></a>Examples
Toggle tracing on and off.

```
trace
```

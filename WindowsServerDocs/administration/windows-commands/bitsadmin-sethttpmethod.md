# bitsadmin sethttpmethod

Sets the HTTP verb to use.

## Syntax

```
bitsadmin /GetHttpMethod <Job> <HTTPMethod>
```

## Parameters

|Parameter|Description|
|---------|-----------|
|Job|The job's display name or GUID|
|HTTPMethod|The HTTP verb to use|

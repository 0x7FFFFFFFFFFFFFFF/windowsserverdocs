# Enable access over a wireless LAN
If you want users to be able to access your Multipoint Services system over a wireless local area network \(LAN\), you must install and enable the Wireless LAN Service. Wireless network adapters are disabled by default in Windows Server. For information about installing and enabling the Wireless LAN Service, see [Wireless LAN Service Overview](http://technet.microsoft.com/library/hh994698.aspx) \(http:\/\/technet.microsoft.com\/library\/hh994698.aspx\).
